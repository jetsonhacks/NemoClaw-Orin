# Local Skills

This repository keeps agent-specific workflow skills under `.agents/skills/`.

These skills are repo-local and are intended to capture recurring debugging and
recovery tasks that are specific to the Jetson/OpenShell/NemoClaw workflow in
this checkout.

Current local skills:

- `nemoclaw-debug-pairing` for recurring OpenClaw pairing triage and recovery
- `nemoclaw-debug-inference-path` for gateway-versus-embedded inference diagnosis
